# 🌿 Tranquil

Tranquil is a simple and responsive web project built using **HTML, CSS, and JavaScript**.  
It focuses on clean UI design, smooth user experience, and lightweight performance.

---

## ✨ Features

- 🎨 Clean and minimal UI design
- ⚡ Fast and lightweight website
- 📱 Fully responsive (mobile + desktop)
- 🧠 Simple and easy-to-understand code structure
- 🌙 Smooth and calm user experience

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---


---

## 🚀 How to Run the Project

You don’t need any installation or setup.

### Option 1: Open directly
1. Download or clone the repository
2. Open `index.html` in your browser

### Option 2: Using VS Code (recommended)
1. Open project in VS Code
2. Install **Live Server extension**
3. Right-click `index.html`
4. Click **"Open with Live Server"**
