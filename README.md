 TRANQUIL – Smart Wellness & Stress Management Web App

 Overview

TRANQUIL is a feature-rich web application designed to help users manage stress, improve mental well-being, and maintain a healthy lifestyle. It integrates multiple wellness tools such as meditation, mood tracking, analytics, and smart recommendations into a single platform.

---

 Features

 Core Modules

* Home Dashboard
* Breathing Exercises
* Relaxation Sounds & Music
* Mood Journal
* Analytics Dashboard
